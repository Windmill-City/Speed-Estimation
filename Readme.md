# Speed-Estimation

行人速度估计

## 环境配置

Python 3.9

```sh
python -m venv venv
venv/Script/activate
```

YoloV8 前置依赖

```sh
pip install lap
```

```sh
pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu117
```

```sh
pip install ultralytics
```

## 运行

```sh
python Track.py
```
